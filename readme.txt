Resumo dos Arquivos

index.php:

Exibe a lista de clientes.
Botões "Cadastrar", "Editar" e "Excluir" para cada cliente.
criar.php:

Formulário para cadastrar um novo cliente.
Processa a inserção no banco de dados.
editar.php:

Formulário de edição para alterar dados de um cliente.
Processa a atualização no banco de dados.
excluir.php:

Processa a exclusão de um cliente.


Fluxo Geral

Na index.php:

O usuário clica em "Cadastrar" para adicionar um novo cliente.
O usuário clica em "Editar" para modificar os dados de um cliente existente.
O usuário clica em "Excluir" para remover um cliente.
O sistema vai redirecionar para o arquivo correspondente (criar.php, editar.php, excluir.php) e realizar a ação desejada.

Esse é o fluxo básico de um CRUD com Cadastrar, Editar e Excluir.
